# BookMeUp

<p>Project that gets the best value per star rating property by searching with dates and client type (regular or Rewards)</p>

<h2>Requirements</h2>

Have python 3.3.3 or higher installed. If you have another version, please consider using [pyenv](https://github.com/pyenv/pyenv) to manage your python versions.

<h2>Usage</h2>

<h3>Command Line Interface</h3>

    $ python main.py
    'Please enter your type of client and the desired dates:'
    $ <client_type>: <date1>, <date2>, <date3>, â€¦
    '<best_hotel>'
